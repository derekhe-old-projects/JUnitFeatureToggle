JUnit Feature Toggle

A JUnit extension which can run test selectively by feature toggle.

Download this project as a .zip file Download this project as a tar.gz file

Welcome

This library is a simple runner which can use FeatureToggle annotation to control the running of test. This could be used when feature need to be turned off. Sample usage of this feature toggle is in the test classes. To build 

$ cd your_repo_root/repo_name
$ gradle build

Use with IntelliJ

$ cd your_repo_root/repo_name
$ gradle idea

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
